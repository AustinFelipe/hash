needsReHash

  • needsReHash(value: string): boolean

  • Implementation of ArgonContract.needsReHash

    • Defined in Drivers/Argon.ts:69

    Returns a boolean telling if the hash needs a rehash or not. The rehash is required when

    1. The argon2 version is changed
    2. Number of iterations are changed.
    3. The memory value is changed.
    4. The parellelism value is changed.
    5. The argon variant is changed.
